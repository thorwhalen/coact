coact.util

Pure helpers for coact with no internal coact imports.

Small, reusable string/name utilities and a thin check_requirements helper for the optional-dependency backends (Agent SDK, py2mcp/FastMCP, an LLM provider). Kept dependency-free so every other coact module can import it.

coact.util.agent_filename(name: str) → str[source]

Return the safe <name>.md filename for an agent, rejecting unsafe names.

Agent names become files under .claude/agents/; a name carrying a path separator or .. could escape that directory (CWE-22 path traversal) on write or read. Names must be a bare filename stem. Raises ValueError on an empty, non-string, or path-bearing name — the check lives at the filesystem boundary so in-memory AgentDefinitions stay unconstrained.

>>> agent_filename('ux-analyst')
'ux-analyst.md'
>>> agent_filename('../escape')
Traceback (most recent call last):
    ...
ValueError: unsafe agent name '../escape': ...
coact.util.check_requirements(modules: dict[str, str], *, feature: str) → None[source]

Verify optional dependencies are importable, else raise an actionable error.

modules maps an importable module name to the pip target that provides it (e.g. {'claude_agent_sdk': 'claude-agent-sdk'}). Raises ImportError listing the exact pip install line for whatever is missing — and, when some requirements are present, which ones — the package-UX convention for optional backends (informative, actionable errors).

>>> check_requirements({'json': 'json'}, feature='noop')  # importable -> no raise
coact.util.first_balanced_span(s: str, opener: str = '{', closer: str = '}') → str | None[source]

Return the first depth-balanced opener…closer substring of s.

Brackets inside double-quoted JSON strings are ignored, so a valid value followed by prose that itself contains brackets is not over-captured (a greedy opener.*closer would run to the last closer and fail to parse). Returns None when no balanced span starts in s. Shared by the LLM-reply JSON extractors (coact.llm, coact.realize_litellm).

>>> first_balanced_span('Result: {"a": 1}. Note: {braces}.')
'{"a": 1}'
>>> first_balanced_span('see [1, [2, 3]] end', '[', ']')
'[1, [2, 3]]'
>>> first_balanced_span('no brackets here') is None
True
coact.util.import_object(ref: str) → Any[source]

Resolve a 'module.path:attr' or 'module.path.attr' reference.

Prefer the module:attr form (unambiguous). Falls back to splitting on the last dot for the dotted form.

>>> import_object('json:dumps')
<function dumps at ...>
>>> import_object('os.path.join')
<function join at ...>
